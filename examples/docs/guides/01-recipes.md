# Working with recipes

A recipe is a reusable template for a brew: method, dose, water weight,
temperature and a grind starting point. Recipes remove the guesswork from
your morning routine — pick one, follow the timer, rate the result.

Create a recipe from any logged brew with **Save as recipe**, or start from
scratch under *Recipes → New*. Brewlog tracks which recipe every brew came
from, so the comparison view can answer questions like "did lowering the
temperature by 2 °C actually improve my washed Ethiopian?".

Recipes are plain JSON files under `~/.brewlog/recipes/` and can be shared
by copying the file.
