# Brewlog

Brewlog is a self-hosted coffee-brewing journal. Log every pour-over, espresso
shot and cold brew with grind size, water temperature and tasting notes, then
track how recipe changes affect the cup — all from a single small binary that
keeps your data on your own machine.

## Highlights

- Runs entirely on your hardware; no accounts, no cloud.
- Import and export plain CSV.
- REST API for integrations.
